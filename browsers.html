Browsers

     Computer hooked up online? Know you need a browser? You have decided on Netscape Navigator or Microsoft Internet Explorer and are ready to start browsing, SO NOW WHAT?

Browser Comparisons

     Can't yet decide on a browser, look at these unbiased comparisons from independent sources.

BrowserWatch
Full site devoted to comparing the various browsers. Has plugins for the latest browsers as well as a page to download whichever browser you decide on.
ThreeToad Browser Comparison
Designed to give you a first hand look at what different browsers see when they come to the same page.
Browser Shootout
Compares many browsers with a 1 to 4 star rating, also has a link on where you can download that browser.

Netscape Navigator

     You understand you have a browser, are connected to the World Wide Web, and that your computer's browser is Netscape Navigator, okay, SO NOW WHAT?

Helpful Setup Tips
A good graphical guide of how to setup your computer for a dial-up Internet connection, and how to setup your Netscape mail in Windows95.
NCTS Graphical Netscape Tutorial
An online graphical tutorial provides user with the basics needed to start surfing the Internet.
Netscape Browser Tutorial
Find a great step-by-step tutorial on how to use Netscape Navigator.
Learn the Net
A friendly, multilingual guide to Internet basics. You can also subscribe to their mailing list.

Microsoft Internet Explorer

     You understand you have a browser, are connected to the World Wide Web, and that your computer's browser is Microsoft Internet Explorer, okay, SO NOW WHAT?

Internet Flying Academy
Explains the basics of Internet Navigation using Microsoft's Internet Explorer. Start with Lesson 1 and gradually work your way to Lesson 12.
Internet Explorer Tutorial
Brief tutorial of how to use the latest version of Internet Explorer. Includes how to use some of it's helper apps like Mail and News.
Microsoft's How To Guide for IE
Demonstrates many of the features of Internet Explorer and shows you how to customize and personalize your copy of the program.

Other Browsers

     You understand you have a browser, are connected to the World Wide Web, and that your computer's browser is either Mosaic, Cyberdog, Opera, or Lynx, okay, SO NOW WHAT?

NCSA Mosaic FAQ's
Answers Frequently Asked Questions about how to use the NCSA Mosaic browser under Windows95/NT.
Cyberdog Documentation
Various sources of information on Apple Computer's Web browser Cyberdog. Includes an online, downloadable manual.
Lynx Users Guide
Full guide for Lynx, a low-end text browser. Covers all the basics and some advanced topics.
